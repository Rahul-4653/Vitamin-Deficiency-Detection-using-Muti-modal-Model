# Vitamin-Deficiency-Detection-using-Muti-modal-Model
AI-powered Multivitamin detection system using CNN and ANN.
Overview

This project is an AI-based health analysis system designed to detect possible vitamin deficiencies using three independent machine learning models:

Skin Analysis Model
Nail Health Model
Symptom Analysis Model

Each model processes a different input modality, and the final prediction is generated by combining their outputs through an application layer.
This system demonstrates a multi-model approach to medical inference using deep learning concepts.

Input Flow:
User uploads image / inputs symptoms
Each model processes input independently
Final output generated in application layer
